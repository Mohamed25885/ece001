SQL stands for Structured Query Language, is a standard programming language to access and manipulate the data. Programmers needed a language or code to be able to handle the data of their projects, that’s why SQL was introduced to handle the data of database of programming project. SQL is a computed based programming language used to deal with data in relational database managing systems. SQL was invented by IBM for querying, and altering relational database using declarative statements. It was first developed at IBM’s San Jose Research Laboratory, first it was known as Sequel language in 1970’s. In 1986, American National Standard Institute published it as SQL, the latest version of it is SQL:2008 updated in 2008
