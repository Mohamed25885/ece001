JAVA is an object-oriented language with runtime environment, it has features and functions combined between C and C++ with the addition of more features, JAVA can be used in web application and software development, that’s why JAVA is one of the general purpose programming languages, JAVA was invented by Sun Microsystems in 1995, It gained its format and syntax from C and its OOP from C++, the programs made by Java are platform independent which means they don’t depend on either the operating system or the processor as long as java interpreter is available. Java can be found in dynamic websites , in mobile applications and games, and even in computer applications.
